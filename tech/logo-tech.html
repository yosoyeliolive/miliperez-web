Reglas de Uso Obligatorias

Área de respeto: Mínimo 20% del alto del logo de espacio libre alrededor. Nunca deformar. Nunca rotar. Nunca cambiar colores fuera de las versiones aprobadas.

Tamaños mínimos:
• Logo completo: 120px de altura mínima
• Logo horizontal: 60px de altura mínima
• Isotipo: 40px de lado mínimo
• Favicon: 32px y 16px

Colores permitidos:
• Azul #008DCB (color principal de marca)
• Blanco #FFFFFF (sobre fondos oscuros)
• Negro #000000 (sobre fondos claros o B/N)
• NUNCA usar grises, degradados o efectos

Tipografía:
• Siempre Inter Weight 100 para "MILI PÉREZ"
• Siempre en mayúsculas
• Siempre con tilde en PÉREZ
• Letter-spacing: 0.2em mínimo

Si el logo no cabe, usa el isotipo. Nunca comprimas el logo horizontal.

Archivos Descargables

Los botones "DESCARGAR SVG" están preparados para generar archivos vectoriales escalables.
Para uso profesional, se recomienda:

• SVG: Para web, aplicaciones digitales (escalable infinitamente)
• PNG: Para redes sociales, presentaciones (exportar en 300dpi)
• PDF: Para imprenta, tarjetas de visita (vectorial para impresión)
• ICO: Para favicons (generar desde versión 32x32 y 16x16)

Este documento HTML contiene todos los SVGs inline. Puedes copiar el código SVG directamente desde el inspector del navegador o usar los botones de descarga.
